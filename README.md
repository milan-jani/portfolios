# 🎨 The Ultimate Portfolio Collection

Welcome to a curated collection of premium, high-performance portfolio templates. Each project is designed with a unique aesthetic and technical focus.

---

## ✨ Static Portfolios (HTML • CSS • JS)
*Quick to deploy, lightweight, and visually stunning.*

### ⚡ **Cyberpunk Neon** (`portfolio-cyberpunk`)
- **Profile:** Creative Developer / Designer
- **Aesthetic:** High-energy neon, glitch effects, and particle backgrounds.
- **Best For:** Developers who want to stand out with a futuristic, bold digital identity.

### 👑 **Royal Luxury** (`portfolio-luxury`)
- **Profile:** Digital Artisan / Premium Consultant
- **Aesthetic:** Gold accents, elegant typography (Cinzel), and refined animations.
- **Best For:** High-end service providers and professionals who value elegance and distinction.

### 🛠️ **Embedded Systems** (`portfolio-embedded`)
- **Profile:** Firmware Engineer / IoT Developer
- **Aesthetic:** Technical, grid-based, system-oriented (JetBrains Mono style).
- **Best For:** Engineering professionals showcasing hardware-software integration.

## 🚀 How to Run

### 📂 **Static Folders**
Just open the `index.html` file in your browser, or run a local server:
`python -m http.server 8080`

### ⚡ **Next.js Folders**
One-time setup for each folder:
1. `npm install`
2. `npm run dev` (Open `http://localhost:3000`)
